# Contributing

Thank you for helping improve VISUAL STUDIO HARNESS.

## Ways to contribute

You can contribute by:

- Reporting issues
- Suggesting improvements
- Submitting pull requests
- Improving documentation

## Contributor Agreement

By submitting a contribution, you agree that:

- You grant the project owner permission to use, modify, include, and distribute
  your contribution as part of this project.
- Your contribution may be included in future versions of the software,
  including commercial versions.

## Guidelines

Please:

- Keep changes focused.
- Explain what problem your change solves.
- Include documentation or tests when appropriate.
- Follow the existing style of the project.
